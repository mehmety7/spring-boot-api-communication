# spring-kafka-application

## Requirements

For building and running the application: you need:

- **JDK 17** 
- **IDE** (i.e. IntelliJ Idea)
- **[Get Docker](https://docs.docker.com/get-docker/)**
- **Apache Maven 3.9.9**

## Running the application locally

- mvnw clean install 
- docker-compose up --build
